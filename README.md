# Steps to follow to Run the weather App
Extract the zip file
1.command to Install necessary requirements pip install uvicorn fastapi streamlit requests
2. To run the Back end: open terminal inside the back-end folder and run this command "python -m uvicorn backend:app --reload"
3. To run the front end: open terminal inside the front-end folder and run this command "python -m streamlit run frontend.py"
